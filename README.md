
### Ts + Vue2.x + vue-router + vuex + axios

简单的摄影小图库

* [] 自动展示照片元数据 (光圈快门ios)
* [] 上传图片自动排版 (时间轴，瀑布流)
* [x] 适配PC与移动版
* [x] 相册基础浏览操作，手势放大
* [x] 图片懒加载
